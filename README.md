# 🛒 Flipkart Clone (Frontend)

A modern e-commerce web application inspired by Flipkart, built using React. This project replicates core shopping functionalities such as product browsing, cart management, wishlist, and checkout flow.

---

## 🚀 Features

### 🛍️ Core Functionality
- Product listing with categories
- Product detail page with image carousel
- Add to Cart & Buy Now
- Wishlist functionality ❤️
- Cart management system
- Order summary & checkout flow

### 📦 Product Enhancements
- Product availability (In Stock / Out of Stock)
- Dynamic stock count display (e.g., "Only 3 left!")
- Product specifications (generated dynamically)

### 🎨 UI/UX
- Flipkart-inspired design
- Clean and modern layout
- Interactive image carousel
- Toast notifications for actions

---

## 🛠️ Tech Stack

### Frontend
- React.js
- React Router DOM
- Axios (API calls)
- CSS (Custom styling)
- React Toastify (notifications)

### Backend (Connected API)
- Node.js
- Express.js
- PostgreSQL
